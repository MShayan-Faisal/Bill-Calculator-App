🧾 Bill Calculator App (React.js)

The Bill Calculator App is a smart and responsive web application built using React.js.
It allows users to calculate the total bill amount, including tips, and even split the bill among multiple people — all with instant real-time updates powered by React’s state management.

Designed with simplicity and modern UI principles, this app demonstrates how React components can be used to build dynamic, reusable, and interactive interfaces efficiently.

🧠 About The Project

The Bill Calculator App was built to make everyday calculations quick, intuitive, and visually appealing.
Whether you’re dining out with friends or managing shared expenses, this app gives you instant, accurate results with minimal input.

It’s designed as a React single-page application (SPA) that reacts instantly to user input without page reloads, thanks to the power of React hooks and component re-rendering.

This project also highlights best practices in React state management, component structuring, and responsive design.

🧰 Technologies Used

⚙️ How It Works

The user enters the bill amount.

They select or enter a tip percentage.

The user specifies how many people will split the bill.

React instantly calculates:

Tip amount

Total amount (bill + tip)

Amount per person

Any change to the inputs automatically re-renders the result in real time.

This functionality is handled entirely through React’s useState hook and event handlers, ensuring smooth, responsive updates.
